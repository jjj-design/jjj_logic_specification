=== 節湯型浴槽について

==== 1. 概要

省エネ基準では節湯型浴槽が評価できない。

自立循環プログラムでは、

- 節湯型浴槽の定義
- 節湯型浴槽による給湯量の削減

を用意する。



<<<
==== 2. 変更箇所

付録J「節湯の効果係数」である。



<<<
==== 3. 変更内容
付録Jに節湯型浴槽の効果係数を追加する。

節湯補正給湯量への適用は、「9．節湯補正給湯量」の式(6d)(6c)において現時点で既に行われている（補正値1）。




<<<
===== 3.1． 付録J
付録Jに以下の効果係数を、表J.1 「節湯の効果係数の適用条件と値」に適用条件を追加する。

[stem]
++++
f_{ sb } = f_{ sb,tub }
++++

ここで、

[cols="<.<1,<.<20", frame=none, grid=none, stripes=none]
|===

|stem:[ f_{ sb } ]
|：浴槽における節湯の効果係数,-

|stem:[ f_{ sb,tub } ]
|：浴槽における節湯型浴槽による節湯の効果係数（= 0.7）,-

|===

である。

表J.1に次の行を追加する

[cols="1,1,1,5,1"]
|===
^.^h|分類
^.^h|節湯の効果 +
係数の種類
^.^h|記号
^.^h|適用条件
^.^h|係数の値

|浴槽
|節湯型浴槽
|stem:[f_{ sb, tub }]
|深さ70%の水量がxx.x[L]以下であるもの
|0.7

|===
